# Kafka-FLY
Segunda etapa do trabalho de Sistemas Orientados a Eventos. Nesta aplicação será implementado o monitoramente do voos por meio de Processamento de Eventos Complexos e uso de Kafka Streams.

Observar a interface do banco de dados, caso queiram:

### Abre 
```bash
http://localhost:8080/h2-console
```

### JDBC URL:
```bash 
jdbc:h2:file:./db/soedb
```

### User Name e Password:
```bash 
deixa vazio
```

### Aperta:
```bash 
connect
```

### Digita no query:
```bash
SHOW TABLES;
```

### Selecionar tabela especifica:
```bash 
SELECT * FROM NOME_TABELA;
```
